# 關於我

嗨，我是 **Bater Chen**，一位擁有超過 12 年經驗的資深全端工程師，專注於在不同產業中提供可擴展的解決方案。我熱衷於透過整潔、可維護的程式碼，運用 AI 為使用者創造真正的價值。

多年來我使用過多種程式語言——從 Java 和 Ruby 開始，目前在工作中主要使用 **C#/.NET** 和 **Python**。我曾在**日本大阪**生活和工作六年，並於 2024 年回到**台灣台北**加入一家新創公司。

過去一年，我在團隊中扮演「救火隊員」的角色——根據需要在前端、後端和專案管理之間切換。目前我領導**後端團隊**，專注於優化系統效能、實施 DevOps 最佳實踐，以及重構程式碼庫使其更加模組化和可維護。

我將 **Sandi Metz** 和 **Kent Beck** 視為我的程式設計導師。我熱衷於**重構遺留程式碼**，讓混亂的專案變得乾淨且易於維護。我相信優秀的軟體始於優秀的開發者體驗（DevEx），並致力於建立支持這一理念的工具和系統。
